Welke site moet ik kiezen?


Geen zorgen! De Online Leerhulp is een gezamenlijk project waarbij wij schoolleerlingen proberen te helpen met informatie over onderwerpen, een eigen chatbot genaamd Arya en nog veel meer!

Instant Games is een Engels-talige website waarmee je simpele spelletjes kan spelen, direct in je webbrowser!

Simple Texteditor is een krachtige online tekstverwerker waarmee je in HTML, CSS en JavaScript websites kan maken en die direct in je browser laten draaien. Als je tevreden bent kan je de bestanden downloaden en ermee doen wat je wilt!

Locker Wallet is een Android-app waarmee je op je Wear OS-smartwatch een NFC-pas kan 'emuleren' en zo toegang kunt krijgen tot een kluis of een ander object dat gebruik maakt van NFC. Op deze site staat een APK-bestand waarmee je Locker Wallet kan installeren. Deze app is momenteel in ontwikkeling en zal nog niet goed werken.
